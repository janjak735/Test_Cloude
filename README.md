# Привет! Добро пожаловать в Claude Code 👋

Это твоя шпаргалка по работе с Claude Code — официальным приложением от Anthropic.

---

## Что такое вкладка Code?

Вкладка **Code** — это умный помощник по программированию прямо у тебя на экране. Ты просто пишешь мне задачу обычными словами, а я пишу код, объясняю его, нахожу ошибки и помогаю разобраться в любом проекте.

---

## Как просить меня писать код?

Пиши задачу так, как тебе удобно. Вот несколько примеров:

- **"Напиши функцию на Python, которая считает сумму списка чисел"**
- **"Сделай простой сайт с кнопкой и счётчиком кликов"**
- **"Объясни, что делает этот код"** — и вставь код следом
- **"Найди ошибку в моём коде"** — покажи мне файл или вставь текст
- **"Добавь в мой проект возможность авторизации"**

Чем точнее ты описываешь задачу, тем лучше я помогу. Но не бойся писать размыто — я уточню, если что-то непонятно.

---

## Полезные команды (начинаются с `/`)

| Команда | Что делает |
|---|---|
| `/help` | Показывает список всех доступных команд |
| `/clear` | Очищает историю чата и начинает заново |
| `/model` | Позволяет переключить модель Claude |
| `/code-review` | Проверяет твой код на ошибки и улучшения |

---

## Советы для удобной работы

1. **Открой нужную папку с проектом** — тогда я смогу читать и редактировать твои файлы напрямую.
2. **Показывай мне файлы** — просто напиши "посмотри на файл `app.py`" и я его прочитаю.
3. **Проси объяснения** — если что-то непонятно в коде, спроси: "объясни это простыми словами".
4. **Итерируй** — не нравится результат? Напиши "переделай, но сделай проще" или "добавь обработку ошибок".
5. **Я помню контекст** — в рамках одного чата я помню всё, что мы обсуждали раньше.
